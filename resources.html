Resources

  • If you know of a website, tutorial, blog post, video, or other resource that you think would be valuable to the class, feel free to email the course staff!

Design


Android

  • Android Design Guide:
    General design patterns and best practices for Android apps.
  • Android Material Design Language:
    Walkthrough of Android's new "Material Design" style, from high-level principles to detailed examples.
  • Android & iOS Design Cheat Sheet:
    Infographic comparing common design patterns and best practices for both Android and iOS apps.

iOS

  • iOS UI Design Guide:
    Design principles and best practices for iOS apps.
  • iOS UI Design Dos and Don'ts:
    Simple illustrated guide for basic iOS design principles. (Targeted for iOS7; most principles are generalizable.)
  • Android & iOS Design Cheat Sheet:
    Infographic comparing common design patterns and best practices for both Android and iOS apps.

Web

  • Bootstrap:
    Popular HTML/CSS/JS framework. Offers nice, modern (but arguably generic) UI elements and animations.
  • Foundation:
    Another popular front-end framework. Great for apps that need to be responsive (different layouts on different platforms).
  • Google WebFonts:
    Massive collection of free fonts to download or add to your web page.
  • Beautiful Web Type:
    A curation of some of the best fonts from Google WebFonts.
  • D3.js:
    A JavaScript library for manipulating webpages based on data. Popular for data visualization / interactive infographics.
  • Animate.css:
    Simple animations done only using CSS.

General

  • Voice And Tone:
    Tips on achieving a consistent and appropriate voice within your app (good vs. bad error messages, etc).
  • Subtle Patterns:
    An awesome collection of subtle background patterns. These can really make your designs pop in a non-distracting way!

Development


Android

  • Android Developers Homepage:
    Tutorials, documentation, and sample code for Android developers.
  • Udacity - Developing Android Apps:
    Google's online course for Android fundamentals. Lecture videos and course materials are free.
  • Stanford CS 193A:
    Stanford's intro Android course.
  • Stack Overflow - Android:
    The Android section of Stack Overflow, a popular question-and-answer platform for coders. Search and ask away!

iOS

  • Apple iOS Developer Center:
    Tutorials, documentation, and sample code for iOS developers.
  • How to make iPhone apps with no programming experience:
    Great beginner's guide to making iPhone apps.
  • Stanford CS 193P Lectures:
    Free lectures from Stanford's intro iPhone course, on iTunes U.
  • Stanford CS 193P Assignments & Example Code:
    Assignments and example code from Stanford's intro iPhone course.
  • Stack Overflow - iOS:
    The iOS section of Stack Overflow, a popular question-and-answer platform for coders. Search and ask away!

Web

  • Codecademy - HTML and CSS:
    Complete beginner's guide to building web pages with HTML and CSS.
  • JS The Right Way:
    Fantastic beginner's guide to coding in JavaScript.
  • Node.js:
    Server-side JavaScript framework and runtime. Get a web server running with 6 lines of code.
  • Node School:
    Interactive walkthroughs help you code your first Node.js application.
  • Ruby on Rails:
    Full-stack web app framework.
  • Ruby on Rails Starter Guide:
    "Getting Started" guide for the Ruby on Rails framework.

Glyphs & Icons


  • The Noun Project:
    Massive collection of glyphs and icons (most are free).
  • One-Div:
    Glyphs and animated icons that use only HTML/CSS (no images).
  • FlatIcon:
    Huge database of free glyphs and icons.

Version Control


  • Understanding GitHub - A Journey For Beginners:
    Great introduction to Git, GitHub, and basic version control principles. Geared towards readers with no background in the subject.
  • GitHubGuides - Hello World:
    Learn how to organize your project using a GitHub repository.
  • TryGit - Learn Git in 15 Minutes:
    Fun, interactive Git tutorial in the browser. Learn the basic principles of version control in 15 minutes.
  • GitHub Foundations:
    Straightforward guide to the basics of GitHub.
  • GitHubGuides YouTube Channel:
    Good video tutorials on how to use GitHub.
